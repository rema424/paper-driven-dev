### 逸脱 #1 (2026-02-20)

- **対象**: §2 仮説（H_main, H_checklist）、§9.1 中止基準
- **内容**: 主仮説を「C > {A, B, D}」から「{C, D} > {A, B} かつ C ≈ D」へピボット。§9.1 の中止基準（「D 条件で C と統計的に同等の co-primary 指標が出た場合: テンプレート効果の主張は成立しない」）が発動したため、論文の thesis を「PDD テンプレートの優位性」から「構造化分析要求の外在化効果」へ方向転換する。
- **理由**: Step 2.2 の統計分析結果が §9.1 の事前登録済み中止基準を満たした。具体的には:
  - 外在化効果 {C, D} > {A, B}: CR p < 0.0001, TP p < 0.0001, Cliff's δ = 1.000（完全分離）
  - フレーミング効果 C > D: 不支持（CR p = 1.000, TP p = 0.576）
  - TOST 同等性 C ≈ D: 確立（CR p < 0.0001, TP p < 0.0001）
- **影響**: 論文の主張構造を全面的に再構築する。主貢献は「構造化要求の外在化が LLM 設計分析品質を劇的に向上させる実験的証拠」に変更。PDD テンプレートは「構造化外在化の一実装」として位置づける。統計分析自体は事前登録どおりに実施済みであり、データの取り扱いに変更はない。ピボットは §9.1 の想定内シナリオに基づく。

### 逸脱 #2 (2026-02-20)

- **対象**: §2 H_framing（副仮説1）
- **内容**: B vs A の TP 有意性（B ≈ 2.0 vs A ≈ 0.2, p < 0.0001, Cliff's δ = 0.900）は事前予測に反する。プロトコル §2 では「B と A の co-primary 2指標は共にゼロ付近で差がない」と予測していたが、TP についてはこれが成立しなかった。
- **理由**: B 条件（論文形式指示）は数学的形式化（証明的記述）を通じてテスト可能な性質を部分的に引き出すことが判明した。これは明示的要求によるものではなく、学術論文フレーミングが間接的に誘導する形式化傾向の副産物と考えられる。
- **影響**: B vs A の TP 有意性は**事後的・探索的発見**（post-hoc exploratory finding）として報告する。事前登録された仮説検定の結果としてではなく、三層階層（A < B < {C, D}）の記述的解釈として §5 Discussion に記載する。この発見は外在化効果の主結論を強化する（B の部分的効果が {C, D} の完全分離と対比される）が、確認的検定の対象としては扱わない。
