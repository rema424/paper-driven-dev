# .internal/

過程で使用した個人メモ・ドラフト・参考資料・ワークショップ成果物を格納する非公開ディレクトリ。

`README.md` のみ Git 追跡対象。他のファイルはすべて `.gitignore` で除外されている。

## ディレクトリ構造

```
.internal/
├── README.md         # このファイル（追跡対象）
├── notes/            # 個人メモ
├── drafts/           # WIP ドキュメント
├── references/       # 外部参考資料の要約
│   ├── subdir1/      # 参考資料1
│   ├── subdir2/      # 参考資料2
│   └── subdir3/      # 参考資料3
└── prompts/          # プロンプト草稿
```
