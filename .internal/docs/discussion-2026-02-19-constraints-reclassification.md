# 議論: constraints disclosed の co-primary 再分類（2026-02-19）

## 問いの本質（Codex 定義）

> 一致率30%と定義境界の不安定性が判明した constraints disclosed を、3指標同等の co-primary として維持しても方法論的厳密性と主張の信頼性が成立するか。

## 初期立場

### CC

**secondary に降格すべき。** 一致率 30% は inter-rater reliability として致命的。降格しても conflicting requirements（80%）と testable properties（50%、厳密定義で B=0 維持・C で完全一致）の2指標で主張の骨格は崩れない。誠実性のシグナルとして査読者の信頼を得る。

### Codex

**secondary に降格すべき。** 一致率30%は co-primary として信頼性不足。不一致原因が構成概念の未確定性を示す。3指標同格だと強い2指標まで方法論全体を疑われるリスク。

## 批評（Codex 主導）

### 合意点

- constraints disclosed の secondary/exploratory 降格
- 降格後も主張骨格は維持可能
- 誠実な開示が査読者信頼に有効

### 論点

1. **outcome switching リスク**: 再採点は事前登録されておらず、結果を見て指標を変更したと解釈される可能性
   - 対策: 変更理由は「効果ではなく測定信頼性」と明示、元の3指標結果も併記
2. **修正コスト**: 編集コストは中、主張再設計コストは中〜高
3. **testable properties 50%**: 厳密定義で B=0 維持・C で完全一致なら条件付き維持可能

## 最終合意

1. constraints disclosed は co-primary から外し、exploratory indicator として再分類する
2. 再分類の理由は「効果の大小」ではなく「測定信頼性（30%一致、定義境界の不安定性）」
3. 主要主張は conflicting requirements と testable properties の2指標で再構成
4. testable properties は「厳密定義固定・B=0 維持・C で一致・感度分析併記」の条件付きで co-primary 維持
5. 3指標の元データは残し、再分類を本文で明示して透明性を担保
